QTI 3 Repository Projection Dictionary

A structured dictionary for the platform's QTI repository model. It describes the QTI package, item, test, interaction, scoring, accessibility, alignment, and artifact fields the platform exposes as SQL and JSON. Full QTI XML remains preserved as the source artifact.

Model scope: Repository, import, search, metadata, alignment, and governance. This is not an assessment delivery/runtime conformance claim.

Generated from dictionary/qti-core.v1.json. The same source also emits SQL comments and OpenAPI field descriptions.
