A structured dictionary for the platform's Caliper event ingestion and projection model. It describes Sensor API envelopes, immutable events, indexed entities, actors, context, profile rules, and governed extensions exposed as SQL and JSON.

Model scope: Event ingestion metadata, immutable raw event retention, search projections, profile/action vocabulary, entity indexing, LTI context joins, and privacy-governed extensions. This is not a formal Caliper certification claim.
